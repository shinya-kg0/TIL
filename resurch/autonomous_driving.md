## 完全自動運転AIの思想：AGI実現へのアプローチ

チューリング株式会社は、**完全自動運転（レベル5）**の実現には、従来の運転支援技術の延長線上ではない、**汎用人工知能（AGI）**的なアプローチが必要であると提唱している。レベル5は条件・地域を限定しないため、車両が自律的に周囲のあらゆる問題を解決できる**堅牢性**と**信頼性**が求められる。

### 1. 完全自動運転の定義と課題

| 項目 | 内容 |
| :--- | :--- |
| **定義** | 運転操作の全部を代替し、人間が運転に一切関与しないレベル。**交通整理員のジェスチャー**や**自然言語による迂回指示**など、人間が行う操作すべてを代替する。 |
| **難易度** | 「何をどう作るか」が未知の**アナーキー（カオス）の領域**に属する。 |
| **課題** | 車両が無限にスケール（増加）した際、人間が介入して問題解決することが困難になるため、車両自身が自律的に諸問題を解決する必要がある。 |

---

### 2. 完全自動運転に必須な要素

完全自動運転を実現するために、「**4つの機能**」「**3つの学習**」「**2つのモデル**」「**1つのシステム**」が必須要素として挙げられる。

#### 4つの機能（人間が行う運転操作の代替）

| 機能 | 役割 | 主要な要件 |
| :--- | :--- | :--- |
| **解釈する（認識）** | 複雑な情報（標識の注釈、交通整理員の手ぶり、乗客の自然言語指示）の認識と文脈理解。 | **自然言語処理**と**文脈理解**。 |
| **想像する（予測）** | 不完全情報（見通しの悪い交差点など）に基づき、見えていない危険を予測しリスク評価を行う。 | 運転経験や常識といった**事前知識の一般化**。 |
| **決断する（計画と決定）** | 無数にある状況に対応するため、決定アルゴリズム自体を学習させる。 | **決定アルゴリズムの学習**、事故発生時の**説明責任**を果たす決断理由の保持。 |
| **交渉する（協調）** | 周囲の車や歩行者と、ウインカーや動きを通じて意思を伝え合い、相手の意図を察する。 | 地域や時代に依存する**コミュニケーション様式の学習**。 |

#### 3つの学習（4機能を支えるAIアプローチ）

1.  **マルチモーダル学習**: 視覚、音声、テキストなど**複数の情報様式を統合的**に学習し、あらゆる入力ソースに対応する。
2.  **画像と自然言語の融合学習（Vision-Language）**: 記号や自然言語に溢れる運転情報を統合的に理解するため、**画像処理と自然言語処理を融合**させ、相互方向の文脈理解を可能にする共通した潜在空間を構築する。
3.  **強化学習**: 予測不能な未知の状況に対応するため、トライアルアンドエラーを通じて複雑な判断能力を**一般化**させる。**環境の内部モデル化（世界モデル）**などのアプローチが有効である。

#### 2つのモデルと1つのシステム

| 項目 | 役割 |
| :--- | :--- |
| **2つのモデル** | **ドライバーモデル**（軽量・高速な運転操作）と**ナビゲーターモデル**（大型・複雑な判断・自然言語解釈）に機能を分離し、効率性と複雑性を両立させる。ナビゲーターモデルが意思決定を行い、ドライバーモデルに指示を出す。 |
| **1つのシステム** | 外部ネットワークやシステムに依存せず、**独立して稼働する単一のシステム**であること。これにより、スケールした際の堅牢性、安全性、信頼性を確保し、車体と融合した設計を実現する。 |

これらの要素は、AIが外部情報を解釈し、自ら考え、行動し、説明し、周囲とコミュニケーションできるという点で、**汎用人工知能（AGI）**の要件を満たすものと捉えられ、完全自動運転はAGI的なものの実現領域であると位置づけられる。

---

### 結論

完全自動運転（レベル5）の実現は、従来の技術の延長ではなく、**4つの機能**、**3つの学習**、**2つのモデル**、**1つのシステム**という構成要素を通じた**AGI的知能**の車載化を目指すという、パラダイムシフトを必要とする高度な挑戦である。
